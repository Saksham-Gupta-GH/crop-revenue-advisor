About AgriSmart

AgriSmart is a machine learning-based crop recommendation system that helps users choose the most profitable crop for a given district, season, and land area.

Built using real-world agricultural datasets covering crop production, rainfall statistics, and market price data across India.

A stacking classifier (Random Forest, Gradient Boosting, AdaBoost) predicts the best crop, while a Gradient Boosting regression model estimates expected revenue.
